# 1 UML 概述

**UML（Unified Modeling Language，统一建模语言）**：一种图形化建模语言，用于描述软件系统的静态结构和动态行为。

UML 共分为 结构图（静态 ) 和 行为图（动态） 两大类

- 结构图（静态 ) 

  ​	类图、对象图、组件图、部署图

- 行为图（动态）

  ​	用例图、顺序图、协作图、状态图、活动图、时序图

- 属性：中间部分

说明：属性和方法前面的“+”、“-”和“#”表示访问级别：

- `+`：public
- `-`：private
- `#`：protected
